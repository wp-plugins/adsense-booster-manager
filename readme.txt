Adsense Booster & Manager Plugin offering facility of Ads Management with ad blocking of specific author's post and specific posts. 
 
== Description ==

**Adsense Booster & Manager** is an Ads management Wordpress plugin. It offers a quicker & flexible way to insert Adsense or any Ads code into a blog post. Besides, it can block the Ads anywhere within a post. It comes with a simple setting page. 

There are numerous plugins for adsense but Adsense Booster & Manager helps you to add ads to the post. If you don't want to have ads on specific page the plugin helps you here by providing the facility to block the ads from some specific posts by using there post ID.
This helps the post to be there in the blog without showing the ads to that post, as this post is most important for boosting the blog traffic. 

Here are some **features** that come with this plugin:

1. The plugin helps to place ads anywhere in the blog posts.
1. There are 3 textareas to copy paste the adsense ad code.
1. Helps to block ads from specific post using there post id's.
1. Block the ads from the posts of a specific author's posts.
1. Set the apperarances where you want to the the ads to be visible.
1. Select required alignment as where to place the ads.

== Installation ==

1. Unzip the downloaded plugin and upload the folder "adsense-booster" to the "/wp-content/plugins/" directory.
1. Go to the plugin menu in WordPress and activate the plugin.
1. Go to the bottom of admin menu and click on "Adsense Booster" and configure the options. Paste the Ads codes to the respective text box.
1. Optionally, add the post ids of posts where you dont want to show the ads(Eg: 1,5,7).
1. Optionally, choose the author from author's dropdown on whose post you don't want to show the ads.
